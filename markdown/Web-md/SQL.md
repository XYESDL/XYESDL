# [数据库教程](https://javaguide.cn/database/basis.html)

## 数据库种类

MySQL、PostgreSQL、Oracle、SQL Server和SQLite

##  数据库CURD语法

```vue
创建数据库、指定参数
CREATE DATABASE mydatabase
  CHARACTER SET utf8mb4
  COLLATE utf8mb4_general_ci;

创建表
CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    ...
);
```
